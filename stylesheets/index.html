Stylesheet Languages

This page lists languages that compile to CSS. It's worth noting that there's also a seperate minifier/formatter available for CSS under minifiers.

less

To use less you must install less:

$ npm install less

Before

.box-shadow(@style, @c) when (iscolor(@c)) {
  box-shadow:         @style @c;
  -webkit-box-shadow: @style @c;
  -moz-box-shadow:    @style @c;
}
.box-shadow(@style, @alpha: 50%) when(isnumber(@alpha)){
  .box-shadow(@style, rgba(0, 0, 0, @alpha));
}
.box { 
  color: saturate(#f938ab, 5%);
  border-color: lighten(#f938ab, 30%);
  div { .box-shadow(0 0 5px, 30%) }
}

After

.box {
  color: #fe33ac;
  border-color: #fdcdea;
}
.box div {
  box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
  -webkit-box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
  -moz-box-shadow: 0 0 5px rgba(0, 0, 0, 0.3);
}

stylus

To use stylus you must install stylus:

$ npm install stylus

Before

border-radius()
  -webkit-border-radius arguments
  -moz-border-radius arguments
  border-radius arguments
  
body
  font 12px Helvetica, Arial, sans-serif
  
a.button
  border-radius(5px)

After

body {
  font: 12px Helvetica, Arial, sans-serif;
}
a.button {
  -webkit-border-radius: 5px;
  -moz-border-radius: 5px;
  border-radius: 5px;
}

sass

To use sass you must install sass:

$ npm install sass

Before

+large
  :font-size 15px
+striped
  tr
    :background #fff
    +large
    &:odd
      :background #000
table
  +striped
  :border none

After

table {
  border: none;}
table tr {
  background: #fff;}
table tr {
  font-size: 15px;}
table tr:odd {
  background: #000;}
