Other Transformers

cdata

cdata just wraps code in cdata tags. You don't need to install anything to use cdata

Before

foo bar

After

<![CDATA[
foo bar
]]>

coffee-script

coffee-script is a high level language that compiles to JavaScript. To use it you will need to install coffee-script:

$ npm install coffee-script

Before

# Assignment:
number   = 42
opposite = true

# Conditions:
number = -42 if opposite

# Functions:
square = (x) -> x * x

# Arrays:
list = [1, 2, 3, 4, 5]

# Objects:
math =
  root:   Math.sqrt
  square: square
  cube:   (x) -> x * square x

After

(function() {
  var list, math, number, opposite, square;

  number = 42;

  opposite = true;

  if (opposite) {
    number = -42;
  }

  square = function(x) {
    return x * x;
  };

  list = [1, 2, 3, 4, 5];

  math = {
    root: Math.sqrt,
    square: square,
    cube: function(x) {
      return x * square(x);
    }
  };

}).call(this);

cson

coffee-script is like JSON but with coffee-script like syntax. To use it you will need to install cson:

$ npm install cson

Before

{
  abc: [
    'a'
    'b'
    'c'
  ]
  a:
    b: 'c'
}

After

{"abc":["a","b","c"],"a":{"b":"c"}}

markdown

markdown is a system for writing documents. The aim is that markdown should be easy to read even when it's in plain text form, but that it can also compile to a nicer formatted html view:

Before

# Heading 1
## Heading 2

```
A code sample
```

 - ItemA
 - ItemB
 - ItemC
 
 1. One
 2. Two
 3. Three

After

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<p><code>
A code sample
</code>

</p>
<ul>
<li>ItemA</li>
<li>ItemB</li>
<li><p>ItemC</p>
</li>
<li><p>One</p>
</li>
<li>Two</li>
<li>Three</li>
</ul>

html2jade

html2jade attempts to convert html back into jade. Jade is a templating language. To use html to jade you will need to install it at version 0.0.7:

$ npm install html2jade@0.0.7

Before

<!doctype html>
<html>
    <head>
      <link rel="stylesheet" href="/site.css" />
      <title>Hello</title>
    </head>
    <body>
      <h1>Hello world!</h1>
    </body>
</html>

After

!!! 5
html
  head
    link(rel='stylesheet', href='/site.css')
    title Hello
  body
    h1 Hello world!

component

The two transformers 'component-js' and 'component-css' produce the JavaScript and CSS files from a component. This can't be easilly demoed here because it requires multiple files to make sense. To use, first install component-builder. 

$ npm install component-builder

Then you can build like any other transformer by pointing it at the component.json file.
