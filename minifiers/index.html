Minifiers

uglify-js

uglify-js parses and minifies JavaScript. To use it you must install uglify-js:

$ npm install uglify-js

Before

function hello(world) {
  return 'hello ' + world;
}

After

function hello(l){return"hello "+l}

If you pass options:{ mangle: false, compress: false, output: { beautify: true } } you can beautify JavaScript instead of minifying it.

Before

function hello(world){return 'hello '+world;}

After

function hello(world) {
    return "hello " + world;
}

uglify-css

uglify-css parses and minifies css. To use it you must install css:

$ npm install css

Before

.foo {
  name: 'foo';
}
#bar {
  name: 'bar';
}

After

.foo{name:'foo'}#bar{name:'bar'}

If you pass an option:{ beautify: true } you can beautify css instead of minifying it.

Before

.foo{name:'foo';}#bar{name:'bar';}

After

.foo {
  name: 'foo'
}

#bar {
  name: 'bar'
}

uglify-json

uglify-json parses and minifies json. You don't need to install anything to use it

Before

{
  "foo": "bar",
  "numbers": [
    1,
    2,
    3,
    4,
    5,
    6,
    7
  ]
}

After

{"foo":"bar","numbers":[1,2,3,4,5,6,7]}

If you pass an option:{ beautify: 2 } you can beautify json (with 2 spaces) instead of minifying it.

Before

{"foo":"bar","numbers":[1,2,3,4,5,6,7]}

After

{
  "foo": "bar",
  "numbers": [
    1,
    2,
    3,
    4,
    5,
    6,
    7
  ]
}
